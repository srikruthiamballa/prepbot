 PrepBot - AI Exam Planner
PrepBot is a smart exam preparation platform designed to help students plan, organize, and optimize their study routine. It uses AI to generate personalized study schedules based on user goals, available time, and exam deadlines.
 Features
 AI-Powered Study Plans
Generates customized daily and weekly study schedules using AI.
 Smart Planner
Automatically distributes subjects and topics based on difficulty and priority.
 Progress Tracking
Track completed tasks and monitor your preparation over time.
 Deadline Optimization
Adjusts study intensity as exams approach.
 Tech Stack
Frontend: React (Vite)
Backend: Node.js / FastAPI
AI Integration: OpenAI API
Database: (MongoDB / SQLite)
 Goal
To simplify exam preparation by removing confusion about "what to study and when", and replacing it with a structured, personalized, and adaptive plan.

 Security Note
API keys and sensitive data are stored securely using environment variables and are not exposed in the frontend.
